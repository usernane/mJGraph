To run the project without any errors do the following:

1 - open jcreator.exe
2 - click file and then click open workspace.
3 - navigate to the project folder and look for the file "Graph3.jcw" and select it.
4 - click open and then run the project.

To run each test alone do the following:
1 - in the top main bar, click "View" and then click "File view"
2 - Then click the "+" sign until you reach the folder "tests".
3 - click on any test file.
4 - in the top main bar, click "Run" and then click "Run File".

To check the API Documents of the project click here: "API Doc".

Current known bugs:
In The class "MinimumSpanningTree.java", if the weight of the edge has a decimal place you might get some problems with the method "compateTo()" The method has some rounding problems, and because of that, the class may not give the correct minimum spanning tree. The same bug applies to the class "ShortestPath.java"
